# ShippingEasy (shippingeasy)

ShippingEasy is an Austin, Texas shipping software platform for online merchants — multi-carrier label printing with discounted USPS, UPS, FedEx, and DHL rates, automated order import and workflow rules, inventory and product management, branded tracking, customer email marketing, and reporting. Acquired by Stamps.com in 2016 and part of the Auctane portfolio that also includes ShipStation, ShipEngine, ShipWorks, MetaPack, Packlink, and Endicia.

## APIs

### ShippingEasy Customer API

HMAC-SHA256 signed REST API for pushing orders into a merchant's ShippingEasy account from a custom marketplace, storefront, or backoffice system that is not already covered by the built-in integration catalog. Operations: list API-enabled stores, create order, find order by ID, find order by external order number, find orders, find orders by store, update order status, and cancel order. For broader multi-carrier label / rate / tracking APIs, ShippingEasy directs developers to sister brand ShipEngine.
